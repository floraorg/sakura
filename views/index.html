sakura

a beautiful, fast, easy to use and open source avatar generator

linear astra floral plain static spectra grid

examples:

/linear/username?text=UN&size=60

/astra/username?text=UN&size=100

/floral/username?text=FL&size=80&bg=ff5500

usage:

/:style/:seed # basic avatar

/:style/:seed?text=XX # custom text

/:style/:seed?size=N # custom size
